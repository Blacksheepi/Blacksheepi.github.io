使用hexo将博客post到github.io

使用fexo和github.io搭建自己的博客

搭建成功后常用的操作

普通流程

  • 进入博客本地文件夹
  • 使用fexo new命令新建文章
  • 使用MarkDown编辑器编辑文章并预览效果
  • 运行hexo clean命令清除缓存（db.json）和generate files
  • 运行hexo generate命令生成静态文件
  • 最后hexo deploy部署文件到网站
